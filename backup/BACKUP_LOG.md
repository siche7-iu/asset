## 되돌리는 방법

아래 4개 파일을 해당 버전 폴더에서 꺼내 원래 위치에 덮어쓰면 됩니다.

```
backup/[버전이름]/index.html  →  index.html
backup/[버전이름]/style.css   →  css/style.css
backup/[버전이름]/app.js      →  js/app.js
backup/[버전이름]/data.js     →  js/data.js
```
