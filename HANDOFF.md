## 2. 지금 동작하는 것 (완료)

### 기본 인프라
- ✅ **순수 HTML + CSS + JS** — 외부 라이브러리·서버 없이 더블클릭만으로 실행
- ✅ **Supabase DB 연동** (`js/db.js`): 자산 **71건** + 이력 **190건**이 Supabase에 저장됨. 앱 시작 시 DB에서 읽어오고, 실패 시 로컬 샘플 데이터로 자동 폴백.
  - DB 연결 상태는 우측 하단 토스트로 표시 (🟢 연결됨 / 🟡 오프라인)
  - **자산 등록 모달**로 추가한 자산은 화면 + DB에 동시 저장 (영구 보존)
- ✅ **샘플 데이터** (`js/data.js`): 자산 71건 로컬 폴백용. 새로고침해도 항상 같은 데이터.
- ✅ **GitHub 백업**: `siche7-iu/asset` 저장소 `main` 브랜치에 최신 코드 올라가 있음.
- ✅ **Playwright 자동 테스트** (`npm test`): 대시보드·목록·상세 화면 스크린샷 + 기본 동작 6개 + AI Agent 4개 테스트.

#### 스크롤 구조
- **`.layout`**: `height: 100vh; overflow: hidden;` — 전체 화면을 꽉 채우는 컨테이너
- **`.sidebar`**: `position: sticky; top: 10px; height: calc(100vh - 20px);` — 왼쪽 떠있는 카드(섬 효과)
- **`.main`**: `overflow: hidden; height: calc(100vh - 20px);` — 오른쪽 콘텐츠 카드
- **`.topbar`**: `position: relative;` — 상단 헤더(자연히 상단 고정)
- **`.content`**: `overflow-y: auto; flex: 1;` — 실제로 스크롤되는 영역(스크롤바는 hover 시에만 표시)
