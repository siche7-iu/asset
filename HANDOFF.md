### 화면 레이아웃 구조

#### 스크롤 구조 (최신)
- **`.layout`**: `height: 100vh; overflow: hidden;` — 전체 화면을 꽉 채우는 컨테이너
- **`.sidebar`**: `position: sticky; top: 10px; height: calc(100vh - 20px);` — 왼쪽 떠있는 카드
- **`.main`**: `overflow: hidden; height: calc(100vh - 20px);` — 오른쪽 콘텐츠 카드
- **`.topbar`**: `position: relative;` — 상단 헤더(스크롤해도 항상 상단에 고정)
- **`.content`**: `overflow-y: auto;` — 실제로 스크롤되는 영역

#### 헤더 (`--header-h: 76px`)
- 왼쪽: 탭 3개(고정자산관리시스템 / 지역별 현황 / 현장 점검)
  - "고정자산관리시스템" 탭은 사이드바 메뉴 클릭 시 항상 활성
  - "지역별 현황", "현장 점검" 탭은 클릭 시 해당 탭 활성화
- 오른쪽: 리포트 생성 버튼 / 알림(종) 아이콘 / 아바타

#### 알림 팝업
- 헤더 오른쪽 벨(종) 아이콘 클릭 → 말풍선 드롭다운 등장
- 알림 3건: "담당 자산 계약 만료 D-14", "점검 기한 초과 1건 발생", "결재 대기 문서 2건"
- 각 알림: **해제**(포커스 스타일 제거), **지우기**(알림 삭제), **더보기** 지원
- **모두 지우기** 버튼 → 전체 비우기, 배지 숨김
- 팝업 바깥 클릭 시 자동 닫힘

#### 사이드바 (최신)
- 상단: NH 빌딩 로고 아이콘(`images/Icon.svg`) + "NH 고정자산관리" 텍스트
- 토글 버튼: 사이드바 우상단 내부(반투명 버튼) → 클릭 시 접기/펼치기
- 활성 메뉴: 흰색 아이콘 박스 + 텍스트 강조
- 대시보드 메뉴: 비활성(`Icon_dashboard.svg`, 흰색) / 활성(`Icon_dashboard2.svg`, 남색) 이중 이미지

### 화면 (5개)

#### 대시보드 (`#/dashboard`)
- **AI 인사이트 배너** + **KPI 6칸**(원형 게이지 포함)
- **내 업무 현황** 카드(할 일 목록) — 오른쪽 사이드 패널
- **노후 위험도 도넛** / **지역별 관리 현황**(한국 지도 + 지역 핀 + 노후도 분석)
- **위험 자산 TOP5** / **최근 이슈 내역** / **계약/점검 일정**
- 진입 애니메이션: 요소 순차 등장 + KPI 숫자 카운터 + 게이지 성장

#### AI Agent (`#/ai-agent`)
- 3열 레이아웃: 추천질문 목록 / 채팅 / 분석 단계·결과 패널
- 추천 질문 클릭 → 자동 분석 진행(분석중 인디케이터 → 6단계 순차 활성화 → 결과 테이블)
- 자산 행 클릭 → 우측 패널이 자산 상세(감가상각 게이지 / 보험 D-day)로 슬라이드 전환

#### 원장관리 — 자산 목록 (`#/list`)
- 표 + 검색창 + 분류 필터. 상단 헤더 검색창과 연동.
- **자산 등록 모달**: `+ 자산 등록` 버튼 → 9개 필드 입력 → 목록 맨 위에 바로 추가

#### 자산 상세 (`#/detail/AST-YYYY-NNNN`)
- 자산 기본 정보 + 감가상각 진행률 게이지 + 담당자 연락처 카드
- 변경 이력 타임라인 + 첨부파일 플레이스홀더

#### 준비 중 (`#/soon`)
- 동작하지 않는 메뉴 클릭 시 "준비 중" 안내 화면

## 3. 파일 구조
```
index.html           화면 (사이드바 + 헤더 탭 + 콘텐츠 섹션)

js/app.js            화면 동작 전부 (라우팅/대시보드/목록·검색/상세/알림팝업)
images/              SVG 아이콘 (Icon.svg, Icon_dashboard.svg, Icon_dashboard2.svg 등)
CLAUDE.md            프로젝트 안내 + 작업 로그 (모든 작업 기록)
DESIGN.md            디자인 시스템 규칙 (단일 기준)
HANDOFF.md           이 문서 — 현재 상태 인수인계
TODO.md              다음에 할 일 목록
tests/               Playwright 테스트 스크립트
```
