### SPA URL 해시 라우팅
메뉴를 클릭하면 브라우저 URL 표시창이 자동으로 바뀝니다:

| 화면 | URL |
|---|---|
| 대시보드 | `index.html#/dashboard` |
| 자산 목록 | `index.html#/list` |
| 자산 상세 | `index.html#/detail/AST-2019-0001` |
| 준비 중 메뉴 | `index.html#/soon` |

- 브라우저 **뒤로/앞으로 버튼**도 동작합니다.
- `file://`로 직접 열어도 안정적으로 동작합니다(해시 방식 사용).
