## 1. 보정 목적

기존 사이드바는 NH 그린 박스 + 파란 활성 아이콘 구성인데, 원본 디자인은 **라인 스타일 홈 아이콘 + 흰색 활성 아이콘 박스** 구성이다.  
원본 디자인을 단일 기준으로 삼아 다섯 영역을 보정한다.

## 2. 보정 항목 5가지 (변경 전 → 변경 후)

### 2-1. 브랜드 로고 — NH 그린 박스 제거, 라인 홈 아이콘으로 교체

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| 좌측 마크 | NH 그린(`#00A651`) 정사각형 36×36 박스 + "NH" 흰 글자 | **라인 스타일 홈(집) 아이콘** (24~26px, 색 `#fff`, stroke 1.8) |
| 우측 텍스트 (1줄) | `<strong>고정자산관리</strong>` | `<strong>NH 고정자산관리</strong>` (NH 부분만 더 굵게: `<b>NH</b> 고정자산관리`) |
| 부제 (2줄) | `Fixed Asset Management` 회색 작은 글자 | 동일 (`#7b97c2` 유지) |
| 정렬 | 로고 36×36 옆 텍스트 | **로고 아이콘 24~26px + 텍스트가 한 줄에 더 가깝게** (gap 10px) |

**CSS 가이드**
- `.brand .logo`를 박스가 아닌 단순 아이콘 래퍼로 축소
  - `width:24px; height:24px;` (또는 padding 없는 인라인 SVG)
  - `background: transparent;` (그린 배경 제거)
  - `border-radius: 0;`
  - 내부 SVG는 색 `#fff`, stroke `1.8`
- `.brand-text strong` font-size 15px / weight 700, `b` 태그 weight 900으로 NH 강조
- 부제 색상은 그대로 (`#7b97c2`)

> ※ 사이트의 일반 룰("NH 그린은 로고 마크에만")은 더 이상 사이드바에 그린 박스가 없는 형태로 변경됨을 기록한다. NH 그린이 필요한 다른 자리는 별도 결정.

### 2-2. 활성 메뉴 (`.nav-item.active`) — 아이콘 박스 색 반전

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| 메뉴 전체 배경 | `rgba(255,255,255,.13)` | 동일 유지 |
| 아이콘 박스 배경 | `--c-primary` (파랑) | **`#FFFFFF`** (흰색) |
| 아이콘 stroke 색 | `#fff` | **`--c-navy`** 또는 `#1E2A3A` (어두운 남색) |
| 그림자 | `0 2px 8px rgba(59,130,246,.45)` | `0 2px 6px rgba(0,0,0,.18)` (은은하게) |
| 텍스트 | `#fff` / 600 | 동일 유지 |

**CSS 변경 위치**: `.nav-item.active .ic`
- `background: #fff;`
- 내부 SVG는 `color: var(--c-navy);` 로 자동 인식되도록 CSS 변수 활용
- 비활성 아이콘은 현재대로 `rgba(255,255,255,.07)` + 흰 stroke 유지

> 비활성 → 활성 전환 시 박스만 흰색이 되고 아이콘은 색 반전되는 효과.

### 2-3. 사이드바 토글 버튼 — 위치·모양 변경

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| 위치 | 사이드바 외부 우측 (`right: -13px`) | **사이드바 내부 우측 상단** (`top: 14px; right: 14px`) |
| 모양 | 원형 26×26 흰 배경 + 보더 | **둥근 사각형 28×20 정도, 반투명 흰 배경** (`rgba(255,255,255,.1)`) |
| 아이콘 | 쉐브론(`>>`) | **이중 패널 아이콘** (사각형 + 좌측 세로 분리선) — 원본 이미지의 토글 아이콘 |
| 색 | 어두운 회색 | `#fff` 80% 투명 |

**CSS 가이드**
- `.sidebar-toggle`을 사이드바 내부 우상단으로 이동
- 박스 그림자 제거 또는 매우 약하게
- 호버 시 배경 `rgba(255,255,255,.18)`

**HTML(T2)**: 토글 버튼 SVG를 이중 패널 아이콘으로 교체  
(`<svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.8"><rect x="3" y="6" width="18" height="12" rx="2"/><line x1="9" y1="6" x2="9" y2="18"/></svg>`)

### 2-4. 메뉴 픽토그램 11개 — 원본 라인 스타일로 통일

원본의 각 메뉴 아이콘 의미와 비교해 SVG를 정비한다. **모두 라인 스타일, stroke 1.8, viewBox 24×24** 기준.

| 메뉴 | 원본 아이콘 의미 | SVG 가이드 |
|---|---|---|
| 대시보드 | 4분할 패널 또는 컨트롤러 | 직사각형 4개 또는 좌측 라인+우측 도트 |
| AI Agent | 로봇 머리 (안테나·눈 2개) | `rect`+`circle 2개`+`line` (현재 유지 OK) |
| 원장관리 | 문서 + 우측 화살표 | `rect`(문서) + `path`(화살표) |
| 리스관리 | 접힌 문서/계약서 | `path`(접힌 모서리) + 가로 라인 3개 |
| 재물조사 | 3개 노드(분자형) | `circle 3개` + `line 3개` 연결 |
| 고정투자예산 | 도넛 차트 | `circle`(외) + `path`(부채꼴 강조) |
| 세무관리 | 달력 + 퍼센트 | `rect`(달력) + 작은 `%` 글자 또는 빗금 |
| 결산관리 | 4점 격자 | `4개 circle` 또는 `4개 rect` |
| 회계감사자료 | 돋보기 + 문서 | `circle`(렌즈) + `line`(손잡이) |
| 기타관리 | 막대 그래프 | `line 3개` 다른 높이 |
| 시스템관리 | 톱니바퀴 | 표준 톱니바퀴 path |

> 현재 구현된 아이콘 중 의미가 일치하면 그대로 두고, 다른 아이콘만 교체한다. **T2가 `index.html`에서 한 번에 교체**.

## 3. 디자인 토큰 / 색상 유지 사항

- `--c-navy` `#002B6C`: 사이드바 배경 (그라데이션 유지 가능)
- `--c-primary` `#3B82F6`: **사이드바 활성 메뉴에서는 더 이상 쓰지 않음** (활성 아이콘 박스 흰색으로 변경됨에 따라). 다른 화면(버튼·차트·강조)에서는 그대로 사용.
- `--c-green` `#00A651`: **사이드바에서 제거**. 다른 곳에서 NH 그린이 필요한 경우 별도 결정. (현재로서는 사용처 없음 → CSS 변수 자체는 보존)
- 흰 글자/회색 글자 톤: 원본 이미지에서 픽한 색 그대로 유지 (`#fff`, `#c2d0e6`, `#7b97c2`, `#6f86ac`)

## 4. 예외 상황 / 결정 사항

1. **접힌 상태(`.sidebar.collapsed`)에서의 활성 아이콘**: 흰색 박스 + 어두운 아이콘 그대로 유지. 텍스트만 숨김.
2. **로고 영역 접힘**: `collapsed` 상태에서는 홈 아이콘만 가운데 정렬로 보이고 "NH 고정자산관리" 텍스트 숨김 (`.brand-text { display: none }` 그대로).
3. **호버 시 색 점프**: 비활성 아이콘 호버 색은 흰색 stroke 유지 (변경 없음). 활성 상태와 시각적 충돌 없음.
4. **하단 알약 링크 길이**: 사이드바 폭 220px에서 텍스트가 잘리지 않도록 padding 8px 12px로 통일.

## 5. 구현 분할

### T1 (CSS) — `css/style.css` 부분 수정
- `.brand .logo`: 배경 제거, 24×24 아이콘 래퍼로 축소
- `.nav-item.active .ic`: 배경 `#fff`, 그림자 톤 다운, 내부 SVG가 어두운 색이 되도록 currentColor 처리
- `.sidebar-toggle`: 위치·모양·색 변경 (사이드바 내부 우상단, 둥근 사각형 반투명)
- `.side-links a`: 둥근 사각형 알약 배경 추가
- (옵션) 미디어쿼리에서 영향 없음 확인

### T2 (HTML 구조 + SVG 아이콘) — `index.html`
- `.brand` 내부 NH 박스를 라인 홈 아이콘 SVG로 교체
- `.brand-text strong`을 `<b>NH</b> 고정자산관리` 구조로 변경 (CSS의 `strong b` 가중치 적용)
- 사이드바 토글 버튼 SVG 교체 (이중 패널 아이콘)
- 메뉴 11개 중 원본과 다른 아이콘 SVG 교체 (위 2-4 표 기준)
- `.side-links` 컨테이너 그대로 두고 텍스트·아이콘만 점검

### T2 (JS) — 변경 없음
- `toggleSidebar()` 함수는 이미 구현됨. 토글 버튼 모양만 바뀌므로 JS 영향 없음.
