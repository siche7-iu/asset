                    영역핵심 요청
                    공통 사항사용자 중심 UI/UX · Pain-Point 해소 · MS Edge 전환 · 신기술 사용환경

                    자산별 프로세스신청·검증 현황판 · 수집 데이터 적정성 검토·사후관리
                    데이터 연동외부 API(시세·주소) · 오류 자동 보정 · 생애주기 관리 · 디지털 이력서

                    제안서 (57p) — UI/UX + 개발사

                    제안서 = RFP 요건의 확장판. 1.1 공통사항(p1~17)이 우리(UI/UX) 파트, 1.2~1.6이 개발사(AI·데이터·연계) 파트.

                    1.1 공통 (우리 UI/UX)사용자 중심 UI/UX 11p · Pain-Point · WebSquare 표준화 · 신기술 고도화
                    1.2 실시간 자산 관리EXAONE+LightGBM AI 관제 · RAG 질의응답 · 긴급 보고
                    1.3 예산·회계결산예산 종합현황판 · 불용예산 AI 탐지 · 감사자료 자동검증
                    1.4 자산별 프로세스신청→검증→승인 현황판 · 외부 API 자동검증(시세·지도·인사DB)
                    1.5 데이터 연동외부 API 연계 · 자산 생애주기 · 디지털 이력서
                    1.6 업무지원프로세스 도식화 · 단계별 체크리스트·템플릿
