## 작업 순서 권장

```
P1 (텍스트 오류 수정, 즉시)
  → P4 (데이터 수치 확인·수정)
    → P2 (CSS 수치 조정)
      → P3 (레이아웃 구조 변경, 별도 세션)
        → P5 (디테일 보정)
```

- **P1 + P4**: 시각적 오류 먼저 제거, 언제든 바로 가능
- **P2**: CSS 수치 변경, 반나절 세션
- **P3**: 구조 변경으로 가장 리스크 있는 작업, 충분한 시간 확보 후 진행
- **P5**: 마무리 다듬기
