### 패널 공통 규칙
- 패널 외부(`.panel`): 배경 `#EEF1F6`, border-radius `16px`, 패딩 `18~20px`
- 패널 내부 흰 카드(`.card-white`): 배경 `#fff`, border-radius `14px`, 패딩 `20px`, 소프트 그림자
- **패널 제목 좌측 강조 막대**: `4px × 16px`, border-radius `2px`, 색 `--c-navy` (수직 중앙 정렬)

### 버튼 (Button)
- **Primary**: 파랑 배경 `--c-primary` + 흰 글자, `height 32px`, `padding 0 16px`, `font-size 13.5px`, `border-radius 8px`
- **Outline**: 흰 배경 + 테두리 `--c-border` + 남색 글자, 동일 크기
- **텍스트 링크**: 파랑 글자, border 없음

### 표 (Table)
- 헤더 행: 연한 회색 배경 + 흐린 글자
- 본문 행: 아래 1px 테두리, hover 시 연파랑 배경 `#f0f6ff`
- 행 클릭 → 상세 화면으로 이동

### 상태 배지 (Badge)
- 알약형 (`border-radius: 999px`)
- 심각: red bg/fg | 경고: amber | 주의: blue | 운영: green | 유휴: purple
