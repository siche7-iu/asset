## 4. 레이아웃 (Layout)

```
┌──────────┬───────────────────────────────────────────────┐
│          │  상단 헤더 (Header) — 흰색, 높이 64px           │
│ 사이드바  ├───────────────────────────────────────────────┤
│ (남색)   │                                                 │
│ 264px    │   콘텐츠 영역 (배경 #F5F5F5)                     │
│ 고정     │   - KPI 카드 줄                                  │
│          │   - 카드 그리드(2~4열)                           │
└──────────┴───────────────────────────────────────────────┘
```
- **사이드바**: 폭 264px, 화면에 고정(스크롤해도 유지), 배경 남색 `--c-navy`.
- **상단 헤더**: 흰색 배경, 아래쪽 1px 테두리 `--c-border`, 높이 64px. 왼쪽=화면 탭, 오른쪽=검색·알림·사용자·주요 버튼.
- **콘텐츠**: 배경 회색, 카드들로 구성. 넓은 화면에서 2~4열 그리드, 좁아지면 1열로.

## 5. 구성요소 규칙 (Components)

### 사이드바 (Sidebar)
- 맨 위: NH 로고 마크(그린) + "고정자산관리 / Fixed Asset Management".
- 메뉴는 **그룹(소제목) + 항목**으로 묶음. 그룹 소제목은 작은 대문자/회색.
- **활성 항목**: 알약형 배경(주요 파랑 `--c-primary`) + 흰색 글자. 비활성: 연한 회색 글자, hover 시 살짝 밝게.
- 맨 아래: 버전 표기, 연계 시스템 안내, 고객 지원, 로그아웃.

### 상단 헤더 (Header)
- 화면 탭(예: 대시보드 / 지역별 현황 / 전망 집계) — 활성 탭은 남색 글자 + 밑줄 강조.
- 오른쪽 **주요 버튼**: 파랑 배경(`--c-primary`) + 흰 글자, 모서리 8px.

### KPI 통계 카드 (Stat Card)
- 흰 카드, 위에 작은 회색 라벨 → 아래 큰 남색 숫자 → (선택) 증감 표시(green▲/red▼).

### 콘텐츠 카드 (Card)
- 흰 배경, 모서리 12px, 은은한 그림자/테두리. 제목(H3) + 내용.

### 표 (Table)
- 머리글 줄: 연한 회색 배경 + 흐린 글자. 본문 줄: 아래 1px 테두리, hover 시 연파랑 배경.
- 행 클릭 → 상세로 이동.

### 상태 배지 (Badge)
- 알약형. 상태 색 규칙(2번 항목) 적용. 예: 운영=green, 점검=amber, 유휴=purple, 폐기=red.

### 버튼 (Button)
- **주요(Primary)**: 파랑 배경 + 흰 글자. **보조(Secondary)**: 흰 배경 + 테두리 + 남색 글자.

### 차트 (Chart)
- 도넛 / 가로 막대 / 누적 막대 / 꺾은선. 색은 4번 팔레트를 순서대로.

### 이력 타임라인 (Timeline)
- 세로선 + 점(주요 파랑). 항목마다 날짜·유형·내용.

## 7. 프로토타입에의 적용 매핑

| 디자인 화면 | 프로토타입 화면 |
|---|---|
| 메인 - 대시보드 | **대시보드** (KPI 카드 + 차트들) |
| 자산 원장 / 시안 표 | **원장관리 = 자산 목록·검색** |
| (자산 클릭 시) | **자산 상세·이력** |

색상·글자·간격·구성요소는 본 문서를 단일 기준(Single Source of Truth)으로 삼는다.
실제 CSS 변수는 `css/style.css` 상단 `:root` 에 위 토큰으로 정의한다.
