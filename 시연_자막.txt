PART 1 — 대시보드

- 고정자산관리시스템 메인 대시보드
- AI가 전체 자산 현황을 자동 분석하여 핵심 인사이트를 제공
- 총 자산 수·취득금액·상태별 현황을 한눈에 확인
- 노후 위험도를 고·중·저로 분류해 분포를 시각화
- 교체 시기가 도래한 자산을 분기별 타임라인으로 표시
- 전국 거점별 자산 현황을 지도에서 직접 확인
- 위험 자산·이슈·계약 만료·노후 현황을 한 화면에서 실시간으로 파악


PART 2 — AI Agent (시나리오 1: 추천 질문 → 자산 상세)

- AI Agent — 자연어로 자산을 조회·분석
- 자주 쓰는 질문을 추천 질문으로 바로 실행
- "본점영업부 노후 PC 현황 분석해줘" 클릭
- AI가 자산 데이터 조회 → 정책 기준 검토 → 위험도 평가 순으로 분석
- 조건에 해당하는 자산 목록과 위험도·교체 비용을 자동 산출
- 자산을 클릭하면 상세 정보로 즉시 이동
- 감가상각 진행률·잔존가액을 게이지로 시각화
- AI 진단 결과와 교체·수리·보고서 등 즉시 실행 가능한 액션 제공


PART 3 — AI Agent (시나리오 2: 직접 질문 → 보고서)

- 자유 형식으로 질문 입력 — "차량 보험 만료 임박 현황 알려줘"
- 보험 계약 조회 → 만기 기준 검토 → 갱신 비용 산정
- 분석 완료 후 보고서 자동 생성 기능 제공
- 클릭 한 번으로 공식 보고서 초안 작성 시작
- 업무 목적에 맞는 보고서 유형 선택
- 요약·현황표·위험 분석·예산 영향·우선 조치·결론까지 자동 완성
- 실무에 바로 활용 가능한 보고서를 AI가 즉시 생성
